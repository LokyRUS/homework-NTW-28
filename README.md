### *Пояснение:*
При соединении дополнительным патч-кордом, протокол STP автоматически блокирует соединение, которое в данный момент для полной связности коммутаторов являются избыточными, переведя его в резерв, для предотвращения образования, так называемой петли (бесконечный режим затопления), тем самым предотвращается широковещательный шторм в сети.
